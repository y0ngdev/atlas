# Atlas

Open-source maps platform. Tile serving, geocoding, routing, and place search.

Atlas gives you the building blocks. You bring the design.

## What you get

- **Map tiles** — Vector tiles you render however you want
- **Geocoding** — Convert addresses to coordinates and back
- **Routing** — Point-to-point routes with turn-by-turn instructions
- **Place search** — Find nearby POIs with distance scoring
- **Trip telemetry** — Collect GPS traces, improve ETAs over time
- **Community corrections** — Users report issues, routing improves
